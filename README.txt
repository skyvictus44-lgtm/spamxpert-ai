
SMS Spam Detector - IBM Fest Edition

Steps:
1. Place spam.csv in this folder.
2. pip install -r requirements.txt
3. python train_model.py
4. python app.py
5. Open http://127.0.0.1:5000

Features:
- NLP (TF-IDF)
- Naive Bayes Classifier
- Accuracy Graph
- Confidence Percentage
- Clean & Secure Input Handling
- Professional UI
